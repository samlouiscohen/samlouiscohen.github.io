"Smells Gud"

Developed at University of Pennsylvania's "Penn Apps", Smells Gud is a webapp that sends users dining hall reccomendations based on their given dietary preferences.

Key features: Built a web scraper to collect live information about Columbia's three main dining halls. Allows users to input both favorite foods and also dietary restrictions (Vegetarian, Glutan-free)

Team size: 2

Technologies used: Python, Flask, HTML/CSS/Javascript, SQLite, Beautiful Soup & Requests (for web-scraping)

"Density"

Built by Columbia's ADI Labs. Our team both improved the front end, and also began working on implementing machine learning techniques in an effort to provide users with density predictions.

Key features: Implemented basic machine learning techniques such as linear regression. Improved UI with new tabs and formatted prediction graphs

Team size: 4

Technologies used: Python, Flask, HTML/CSS/Javascript, Pandas, NumPy
